# Landing Page - Lic. Silvia Quiroz (Servicios de Enfermería)

Una landing page profesional diseñada con un enfoque Mobile-First, altamente optimizada para la conversión (UX/UI) y enfocada en generar una conexión directa y sin fricciones vía WhatsApp para la contratación de servicios de salud integrales a domicilio. El proyecto prioriza la estética premium, la limpieza visual y la construcción inmediata de credibilidad.

## Tecnologías Usadas (Tech Stack)

Este proyecto está construido con herramientas modernas enfocadas en el rendimiento y la experiencia de desarrollo a la vanguardia frontend:

*   **React:** Biblioteca de JavaScript para construir interfaces de usuario de alta calidad.
*   **Vite:** Entorno de desarrollo frontend ultrarrápido y empaquetador eficiente.
*   **Tailwind CSS:** Framework CSS utility-first para un diseño ágil, responsivo y escalable sin abandonar el HTML.
*   **Framer Motion:** Biblioteca para animaciones fluidas, micro-interacciones (fricción, hover, tap) e impacto visual.
*   **Despliegue Continuo / Hosting:** Vercel (Optimizado).

## Características Principales (Features)

*   **Diseño Responsivo y Moderno**: Experiencia impecable dictada por una métrica 'Mobile-First', con adaptación fluida y estilización premium en pantallas medianas y de escritorio (ej: 'Sellos de Credibilidad' escalables).
*   **Integración Directa (Call to Action)**: Formularios de cero fricción, reemplazados con botones directos hacia WhatsApp pre-configurados para aumentar radicalmente la tasa de contacto de pacientes.
*   **Componentes Reutilizables de Alta Fidelidad**: Tarjetas de servicio (Service Cards) dinámicas, con diseño neumórfico sutil, que ofrecen efectos inmersivos de hover (escritorio) y respuesta táctil (móvil).
*   **Optimización Activa**: Jerarquías de texto precisas, accesibilidad en el manejo de colores de contraste e imágenes sin fondo (blend-modes) integradas al entorno cromático nativo del sitio.

## Instrucciones para Servidor Local

Sigue estos comandos para correr y modificar el proyecto de manera local en tu entorno de desarrollo:

1. Clona este repositorio o descarga los archivos localmente.
2. Abre la terminal en el directorio del proyecto y asegúrate de tener [Node.js](https://nodejs.org/) instalado.
3. Instala todas las dependencias requeridas del proyecto:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo local:
   ```bash
   npm run dev
   ```
5. Abre la URI proporcionada en tu navegador (usualmente `http://localhost:5173/`).
